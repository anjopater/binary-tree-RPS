# RPS tournament in JavaScript

Gived an array.

    const inputAnswers = [
      [
        [
            [["ARMANDO", "P"], ["DAVE", "S"]],
            [["RICHAR", "R"], ["MICHAEL", "S"]]
        ],
        [
            [["ALLEN", "S"], ["OMER", "P"]],
            [["DAVID", "R"], ["Richard x", "P"]]
        ],
      ]
    ];

Will return the winner = `["RICHAR", "R"]`

## usage
1. Copy the code of the file `binary-rps-tournament.js` in [ES6 Console](https://es6console.com/) and run it.
2. Or Choose your V8 engine prefered.
